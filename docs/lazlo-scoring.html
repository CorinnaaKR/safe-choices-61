Issues to fix
⚠️ Issue 1 — Pass mark is 75%, spec says 85%
PASS_THRESHOLD = 75 in src/pages/ResultsPage.tsx. Change to 85. At 85%, the coffee→validate→sounds-good→ACT-Early path (89%) passes comfortably; the symbol-then-trusted-adult path (79%) fails — which is appropriate.
⚠️ Issue 2 — Evidence not counted; above-100% is impossible
Evidence items total ~150 pts but are excluded from totalPoints in useSimulation.ts. To allow above-100%, evidence points need to be added to the numerator while the denominator stays as path-max-decisions. A thorough player who collects all evidence on the optimal path would score ~314% (70 + 150 decision-evidence points ÷ 70). Needs a cap or weight decision before implementing.
